# Zendo

A simple, calm space for managing your daily tasks.

Zendo is a full-stack to-do application built with the MERN stack. It focuses on clarity and ease of use, allowing you to create, organize, and manage tasks without distraction.

---

## Features

* Create, update, and delete tasks
* Clean and minimal interface
* User accounts with secure authentication
* Persistent data storage
* Drag and drop to reorder tasks
* Dark mode support

---

## Authentication

Zendo includes a complete authentication system:

* User registration and login
* Passwords securely hashed using bcrypt
* JWT-based authentication
* Cookie-based session handling

---

## Technologies Used

### Hosting
* Vercel (Frontend and API)
* MongDB Atlas (Database)
### Frontend

* React 19
* Vite
* Axios
* React DOM 19
* ESLint

### Backend

* Node.js
* Express 5
* MongoDB (Mongoose 9)
* Bcrypt
* JSON Web Tokens (JWT)
* Cookie-parser
* CORS
* Dotenv
* Nodemon

---

## Philosophy

Zendo is designed to be simple, and focused.
No clutter, no unnecessary features, just a space to keep track of what matters.
